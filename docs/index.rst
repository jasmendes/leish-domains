Features
--------

* **Dual Interface**: Both GUI and CLI modes for different workflows
* **Data Processing**: Merge, filter, and analyze protein domain data
* **Statistical Analysis**: Comprehensive statistical tools and visualizations
* **Session Management**: Save and load analysis sessions
* **Excel Export**: Convert results to Excel format
* **Extensible**: Modular design for easy customization

Quick Start
-----------

Install LeishDomains:

.. code-block:: bash

   pip install leishdomains

Run in GUI mode:

.. code-block:: bash

   leishdomains

Run in CLI mode:

.. code-block:: bash

   leishdomains merge file1.txt file2.txt -o merged.txt
